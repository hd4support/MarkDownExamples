#### Heading/Titles 

```
# Heading H1  #######
## Heading H2 #######
### Heading H3 ######
#### Heading H4 #####
##### Heading H5 ####
###### Heading H6 ###

Alt-H1
=======

Alt-H2
------
```

Translates to :

# Heading H1  #######
## Heading H2 #######
### Heading H3 ######
#### Heading H4 #####
##### Heading H5 ####
###### Heading H6 ###


Alt-H1
=======

------

#### Text emphasis


``` Markdown
Bold: **Double Asterisks**   __Double Underlining__

Italic: *Single Asterisk*  _Single Underline_

Strikethrough: ~~Double tilde~~

Bold Italic : **Bold __Italic__**
```

Translates to 

Bold: **Double Asterisks**   __Double Underlining__

Italic: *Single Asterisk*  _Single Underline_

Strikethrough: ~~Double tilde~~

Bold Italic : **Bold _Italic_**
