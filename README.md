
# MarkDown Examples


#### Heading/Titles 
```
# Heading H1  #######
## Heading H2 #######
### Heading H3 ######
#### Heading H4 #####
##### Heading H5 ####
###### Heading H6 ###
```

Translates to :

# Heading H1  #######
## Heading H2 #######
### Heading H3 ######
#### Heading H4 #####
##### Heading H5 ####
###### Heading H6 ###
