
# MARKDOWN EXAMPLES 

#### Heading/Titles 

```
# Heading H1  #######
## Heading H2 #######
### Heading H3 ######
#### Heading H4 #####
##### Heading H5 ####
###### Heading H6 ###
```

Translates to :

# Heading H1  #######
## Heading H2 #######
### Heading H3 ######
#### Heading H4 #####
##### Heading H5 ####
###### Heading H6 ###


#### Text emphasis


``` Markdown
Bold:

**Double Asterisks**

__Double Underlining__

Italic:

*Single Asterisk* 

_Single Underline_

Strikethrough:

~~Double tilde~~
```

Translates to 

Bold:

**Double Asterisks**

__Double Underlining__

Italic:

*Single Asterisk* 

_Single Underline_

Strikethrough:

~~Double Tilde~~
